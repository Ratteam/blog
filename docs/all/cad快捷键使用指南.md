# CAD快捷键命令大全
## 符号键（CTRL开头）
CTRL+1 PROPCLOSEOROPEN 对象特性管理器
CTRL+2或4 ADCENTER 设计中心
CTRL+3 CTOOLPALETTES 工具选项板
CTRL+8或QC QuickCalc 快速计算器

CTRL+N或N New 新建
CTRL+O OPEN 打开
CTRL+P PRINT 打印

CTRL+S或SA QSAVE或SAve 保存

CTRL+V PASTECLIP 粘贴
CTRL+X CUTCLIP 剪切
CTRL+Y REDO 重做
CTRL+Z U 放弃
CTRL+[ Cancels
